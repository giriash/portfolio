Rwanda Lakeside Fish Farm Website

This official website provides information about Rwanda lakeside fish farm to people who want visit or know about there. The client had an old version official website but they want to make it more engaging, relevant and useful to their diverse audiences. They also want to add and share data via the website.

To achieve this, we redesign their website and restructure it to improve user experience and functionalities. Now the website enables users to login, sign up, ask questions and other functionalities. Only fish farm managers can answer online questions and manage user information on the website through invited code login. The data is loaded and managed by SQL language.
