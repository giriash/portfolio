level design

Level generate: based on “building blocks”:

Level generate by building blocks: (By Andrew Aquino): To streamline our terrain generation in a manual fashion, we chose to create each level through ‘building blocks’. A level editor will generate a building block that consists of these attributes.


      1. A list of polygon structures.
      2. A list of center coordinates to initialize physics objects.
      3. An input type.
      4. An output type.
      5. Frequency/Weight (Rare ones get less weight)


The left figure shows the possible building blocks that could be made with dynamic physics. Moreover, it also shows how the token-style building blocks would fit together. The right building blocks shows more static environments.

Problem & Improvement:
