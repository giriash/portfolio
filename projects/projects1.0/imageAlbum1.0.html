Personal Image Album Website

This website is built for people who needs a website to show their photopraphs. It was developed by HTML/CSS, Javascript, Jquery, PHP and other widely used web development programming languages. PHP and SQL languages are used to interact with MySQL database and loaded data. In this website, all visotors can search and interact with photos and albums on the website. Login funtionality enables registered user to manage photos and albums. They can add, delete, edit photos and albums on the website and database will change automatically.


1. Photos View

Both registered and unregistered could access to this page and view all photos on the website. They can click on each photo to get a better view and all information about currrent photo.

2. Albums View

Both registered and unregistered could access to this page and view all albums on the website. They can click on each album to find photos in current album.

3. Search Content

Both registered and unregistered can search key words. And it will search the input key word in all database and show corresbonding results.

4. Login

Registered information has been stored in database. Users can only edit photos and ablums after login. The website will also remind unlogin users to login when they click edit function buttons

  • delete a photo from one album or database
